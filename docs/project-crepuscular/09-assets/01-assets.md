## Asset Rules

- Use SVG whenever possible.
- Use PNG only for raster imagery.
- Keep every master asset editable.
- Every exported asset references its source.
- Astronomical imagery should be scientifically credible.
- Decorative assets never compete with observation data.

## Naming

Examples:

- vesper-logo.svg
- hero-horizon-v1.png
- mars-card.webp
- eclipse-totality-hero.png
- onboarding-step-01.png
- bg-space-primary.webp

## Export Targets

- iOS @1x @2x @3x
- Android mdpi–xxxhdpi
- WebP for large backgrounds
- SVG for icons and vectors
