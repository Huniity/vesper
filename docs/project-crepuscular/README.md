# Project Crepuscular

The design system behind **Vesper**.

## North Star

> Vesper does not try to keep people inside the app. It brings them back to the sky.

Vesper turns the changing night sky into a shared daily discovery. It respects two rare things equally: **the night** and **the user's attention**.

## Four Pillars

1. **Wonder** — preserve the feeling of discovery.
2. **Precision** — measured, not guessed.
3. **Calm** — silence and restraint are features.
4. **Discovery** — every session should reveal something worth remembering or sharing.

## Product Standard

Every screen, notification, animation, component, and piece of copy must answer:

- Does it respect the night?
- Does it respect the user's attention?
- Does it strengthen at least one pillar?
- Does it help the user look up?
- Does it feel unmistakably like Vesper?
