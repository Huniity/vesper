# Components

**Status:** Approved  
**Version:** 1.0

This chapter translates Project Crepuscular into reusable interface parts.

Components must be:

- accessible
- theme-aware
- responsive
- localized
- usable in red night mode
- compatible with Reduced Motion
- documented with all states
- composable without visual drift

## Required Component Documentation

Every production component must define:

- purpose
- anatomy
- variants
- states
- responsive behavior
- content rules
- accessibility behavior
- red night-mode behavior
- motion behavior
- implementation notes
- anti-patterns
